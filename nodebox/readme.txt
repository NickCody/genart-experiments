Welcome to genart-experiments NodeBox samples.

2009-11-21

To work with these samples you need to be running on a Mac capable of running the NodeBox software. You can read more about NodeBox and download it here:

http://nodebox.net/code/index.php/Home

NodeBox is a graphics environment based on the Python programming language. The sample code contained here will render generative art when loaded and run within the NodeBox IDE.
